## **Теоретичні відомості**

### **Способи підключення CSS**

CSS можна підключити до HTML трьома способами. Зовнішній файл є рекомендованим підходом: стилі зберігаються окремо та можуть використовуватися на багатьох сторінках одночасно.

```html
<!-- 1. Зовнішній файл (рекомендовано) -->
<link rel="stylesheet" href="style.css" />

<!-- 2. Внутрішній блок стилів -->
<style>
  h1 {
    color: #1565c0;
  }
</style>

<!-- 3. Атрибут style (інлайн) — використовується лише в крайніх випадках -->
<p style="color: red;">Текст</p>
```

### **Типи селекторів**

Селектори визначають, до яких елементів застосовуються CSS-правила.

```css
p {
  line-height: 1.6;
} /* елемент — всі <p> */
.card {
  border-radius: 8px;
} /* клас — всі елементи з class="card" */
#header {
  background: #1565c0;
} /* ідентифікатор — елемент з id="header" */
.card h2 {
  font-size: 1.25rem;
} /* нащадковий — <h2> всередині .card */
a:hover {
  text-decoration: underline;
} /* псевдоклас — при наведенні */
tr:nth-child(even) {
  background: #f5f5f5;
} /* псевдоклас — парні рядки */
```

### **Специфічність та каскадність**

Якщо кілька правил конкурують за один елемент, браузер застосовує правило з найвищою специфічністю:

| Тип селектора             | Вага |
| ------------------------- | ---- |
| Інлайн-стиль (`style=""`) | 1000 |
| Ідентифікатор (`#id`)     | 100  |
| Клас, псевдоклас          | 10   |
| Елемент, псевдоелемент    | 1    |

```css
p {
  color: black;
} /* специфічність: 1  */
.note {
  color: orange;
} /* специфічність: 10 — переможе */
#warning {
  color: red;
} /* специфічність: 100 — переможе обидва */
```

При однаковій специфічності перемагає правило, оголошене **пізніше** у CSS-файлі.

### **Кольори, шрифти та одиниці виміру**

```css
/* Формати кольорів */
.a {
  color: tomato;
} /* ключове слово */
.b {
  color: #1565c0;
} /* HEX */
.c {
  color: rgb(21, 101, 192);
} /* RGB */
.d {
  background: rgba(0, 0, 0, 0.5);
} /* RGBA — прозорість від 0 до 1 */

/* Властивості шрифтів та тексту */
body {
  font-family:
    "Segoe UI", Arial, sans-serif; /* список з запасними варіантами */
  font-size: 16px;
  font-weight: 400; /* 100–900, або normal / bold */
  line-height: 1.6;
  color: #212121;
}

h1 {
  text-align: center; /* left | center | right | justify */
  text-transform: uppercase; /* none | uppercase | lowercase | capitalize */
  text-decoration: none; /* none | underline | overline | line-through */
}
```

Найпоширеніші одиниці виміру: `px` — фіксований розмір у пікселях, `%` — відсоток від батьківського елемента, `rem` — відносно розміру шрифту кореневого елемента.

### **CSS-змінні (Custom Properties)**

CSS-змінні зберігають значення, що повторюються, в одному місці. Щоб змінити основний колір по всьому проєкту, достатньо відредагувати одну змінну.

```css
:root {
  --color-primary: #1565c0;
  --color-bg: #f5f5f5;
  --font-size-base: 16px;
  --border-radius: 8px;
}

.card {
  background-color: var(--color-bg);
  border-radius: var(--border-radius);
}
```

#### **Завдання 4: Стилізування структури сторінки**

Оформіть `<header>`, `<section>`, списки та `<footer>`.

**Вимоги:**

- `<header>` — колір фону з CSS-змінної, відступи, заголовок по центру, колір тексту
- `<section>` — відступи між секціями, роздільна лінія (`border-bottom`)
- `<ul>` та `<ol>` — змінити стандартний маркер/нумерацію, задати відступи між пунктами
- `<footer>` — темний фон, текст по центру, розмір шрифту менший за базовий

**Приклад структури:**

```css
header {
  background-color: var(--color-primary);
  color: #ffffff; /* замініть на колір тексту */
  padding: 32px 20px; /* замініть на потрібне значення */
  text-align: center;
}

section {
  padding: 24px 0; /* замініть на потрібне значення */
  border-bottom: 1px solid #e0e0e0; /* замініть на потрібну лінію */
}

ul {
  list-style-type: square; /* square | disc | circle | none */
  padding-left: 24px; /* замініть на потрібне значення */
}

ul li,
ol li {
  margin-bottom: 6px; /* замініть на потрібний відступ */
}

footer {
  background-color: #212121; /* замініть на темний колір */
  color: #bdbdbd; /* замініть на світлий колір тексту */
  text-align: center;
  padding: 20px; /* замініть на потрібне значення */
  font-size: 0.875rem; /* менше за базовий — замініть за потреби */
}
```

### **Етап 3: Перевірка**

3.1. Відкрийте `index.html` у браузері та переконайтесь, що стилі застосовуються коректно.

3.2. Відкрийте DevTools (F12) → вкладка **Elements**. Виберіть будь-який елемент і перегляньте панель **Styles** — мають бути видні застосовані правила, перекриті відображаються закресленими.

3.3. У панелі **Styles** виберіть елемент `<html>` у дереві Elements — у списку стилів з'явиться блок `:root { }`, де мають відображатись усі оголошені CSS-змінні.

3.4. Перевірте валідність CSS за допомогою валідатора: https://jigsaw.w3.org/css-validator/#validate_by_input

## **Типові помилки та їх вирішення**

| Проблема                                | Рішення                                                                             |
| --------------------------------------- | ----------------------------------------------------------------------------------- |
| Стилі не застосовуються до сторінки     | Перевірте підключення `<link>` у `<head>` та правильність шляху до `style.css`      |
| CSS-змінна не працює                    | Змінна має бути оголошена у `:root {}` і починатися з двох дефісів `--`             |
| Правило не діє, хоча написане правильно | Відкрийте DevTools → Styles і знайдіть правило з вищою специфічністю, що перекриває |
| Шрифт не змінюється на всій сторінці    | Задайте `font-family` для `body` — він успадковується всіма вкладеними елементами   |
| Таблиця має подвійні рамки              | Додайте `border-collapse: collapse` до правила для `<table>`                        |
