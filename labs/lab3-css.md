# **Лабораторна робота №3**

**Тема:** Базове стилізування CSS

**Мета:** Закріпити практичні навички підключення та написання CSS: використання різних типів селекторів, задання кольорів та шрифтів, робота з каскадністю та специфічністю, застосування CSS-змінних для організації стилів.

**Обладнання:** IBM-сумісні ПК

**Програмне забезпечення:** ОС Windows 10/11, macOS або Linux; веб-браузер (Chrome, Firefox, Edge); текстовий редактор (VS Code, Sublime Text або інший)

## **Контрольні запитання**

1. Які є три способи підключення CSS до HTML-документа? Який із них є рекомендованим і чому?
2. Що таке специфічність (specificity) у CSS? Як браузер вирішує, який стиль застосувати, якщо кілька правил конкурують за один елемент?
3. Яка різниця між класовим селектором (`.class`) та селектором ідентифікатора (`#id`)? Коли доцільно використовувати кожен із них?
4. Що таке CSS-змінні (custom properties)? Як їх оголосити та використати?
5. Що таке успадкування у CSS? Які властивості успадковуються дочірніми елементами, а які — ні?

## **Теоретичні відомості**

### **Способи підключення CSS**

CSS можна підключити до HTML трьома способами. Зовнішній файл є рекомендованим підходом: стилі зберігаються окремо та можуть використовуватися на багатьох сторінках одночасно.

```html
<!-- 1. Зовнішній файл (рекомендовано) -->
<link rel="stylesheet" href="style.css" />

<!-- 2. Внутрішній блок стилів -->
<style>
  h1 {
    color: #1565c0;
  }
</style>

<!-- 3. Атрибут style (інлайн) — використовується лише в крайніх випадках -->
<p style="color: red;">Текст</p>
```

### **Типи селекторів**

Селектори визначають, до яких елементів застосовуються CSS-правила.

```css
/* Селектор елемента — всі <p> на сторінці */
p {
  line-height: 1.6;
}

/* Класовий селектор — всі елементи з class="card" */
.card {
  border-radius: 8px;
}

/* Селектор ідентифікатора — єдиний елемент з id="header" */
#header {
  background: #1565c0;
}

/* Нащадковий: <h2> всередині .card */
.card h2 {
  font-size: 1.25rem;
}

/* Псевдоклас: посилання при наведенні */
a:hover {
  text-decoration: underline;
}

/* Псевдоклас: кожен парний рядок таблиці */
tr:nth-child(even) {
  background-color: #f5f5f5;
}
```

### **Специфічність та каскадність**

Якщо кілька правил конкурують за один елемент, браузер застосовує правило з найвищою специфічністю. Специфічність визначається типом селектора:

| Тип селектора             | Вага |
| ------------------------- | ---- |
| Інлайн-стиль (`style=""`) | 1000 |
| Ідентифікатор (`#id`)     | 100  |
| Клас, псевдоклас, атрибут | 10   |
| Елемент, псевдоелемент    | 1    |

```css
p {
  color: black;
} /* специфічність: 1  */
.note {
  color: orange;
} /* специфічність: 10 — переможе */
#warning {
  color: red;
} /* специфічність: 100 — переможе обидва */
```

При однаковій специфічності перемагає правило, оголошене **пізніше** у CSS-файлі.

### **Кольори та одиниці виміру**

CSS підтримує різні формати запису кольорів:

```css
/* Усі записи рівнозначні — це різні формати одного кольору */
.title {
  color: tomato;
} /* ключове слово */
.title {
  color: #e74c3c;
} /* HEX */
.title {
  color: rgb(231, 76, 60);
} /* RGB */
.overlay {
  background: rgba(0, 0, 0, 0.5);
} /* RGBA — прозорість від 0 до 1 */
.heading {
  color: hsl(214, 80%, 42%);
} /* HSL */
```

Найпоширеніші одиниці виміру розмірів:

| Одиниця | Тип       | Пояснення                                   |
| ------- | --------- | ------------------------------------------- |
| `px`    | Абсолютна | Пікселі — фіксований розмір                 |
| `%`     | Відносна  | Відсоток від батьківського елемента         |
| `em`    | Відносна  | Відносно розміру шрифту поточного елемента  |
| `rem`   | Відносна  | Відносно розміру шрифту кореневого елемента |

### **Властивості шрифтів та тексту**

```css
body {
  font-family:
    "Segoe UI", Arial, sans-serif; /* список шрифтів із запасними варіантами */
  font-size: 16px;
  font-weight: 400; /* 100–900, або normal / bold */
  font-style: italic;
  line-height: 1.6; /* міжрядковий інтервал */
  color: #212121;
}

h1 {
  text-align: center; /* left | center | right | justify */
  letter-spacing: 0.05em; /* відстань між літерами */
  text-transform: uppercase; /* none | uppercase | lowercase | capitalize */
  text-decoration: none; /* none | underline | line-through */
}
```

### **CSS-змінні (Custom Properties)**

CSS-змінні зберігають значення, що повторюються, в одному місці. Це спрощує підтримку стилів: щоб змінити основний колір по всьому проєкту, достатньо відредагувати одну змінну.

```css
/* Оголошення змінних у кореневому елементі */
:root {
  --color-primary: #1565c0;
  --color-bg: #f5f5f5;
  --font-size-base: 16px;
  --border-radius: 8px;
  --spacing-md: 16px;
}

/* Використання змінних */
.button {
  background-color: var(--color-primary);
  padding: var(--spacing-md);
  border-radius: var(--border-radius);
}
```

### **Успадкування**

Деякі CSS-властивості автоматично передаються дочірнім елементам від батьківських. Властивості тексту (`color`, `font-family`, `font-size`, `line-height`) — успадковуються. Властивості блокової моделі (`margin`, `padding`, `border`, `width`) — ні.

```css
body {
  font-family: Arial, sans-serif;
  color: #333; /* успадкується всіма вкладеними елементами */
}

.child {
  color: inherit; /* явне успадкування значення від батька */
}
```

### **Етап 2: Виконання завдань**

---

#### **Завдання 1: CSS-змінні та базові стилі**

Оголосіть CSS-змінні для всього проєкту та задайте базові стилі документа.

**Вимоги:**

- Оголосити у `:root` щонайменше 5 змінних: основний колір, акцентний колір, колір тексту, базовий розмір шрифту та значення border-radius
- Задати `font-family`, `font-size`, `color` та `line-height` для `body` з використанням оголошених змінних
- Обмежити максимальну ширину сторінки та вирівняти вміст по центру за допомогою `margin: 0 auto`
- Задати стилі для заголовків `h1`, `h2`, `h3`: колір, розмір, відступи

---

#### **Завдання 2: Стилізування навігації та посилань**

Оформіть блок навігації та всі посилання на сторінці.

**Вимоги:**

- Задати колір фону та відступи для `<nav>`
- Стилізувати посилання всередині навігації: прибрати стандартне підкреслення, задати колір
- Додати псевдоклас `a:hover` — зміна зовнішнього вигляду посилання при наведенні курсора
- Додати псевдоклас `a:visited` — відмінний колір для відвіданих посилань

---

#### **Завдання 3: Стилізування списків**

Змініть зовнішній вигляд ненумерованого та нумерованого списків.

**Вимоги:**

- Для `<ul>` — змінити або прибрати стандартний маркер за допомогою `list-style-type`
- Для `<ol>` — змінити відступи або стиль нумерації
- Задати відступи між пунктами списку за допомогою `margin`

---

#### **Завдання 4: Стилізування таблиці**

Оформіть HTML-таблицю так, щоб вона була читабельною та акуратною.

**Вимоги:**

- Задати `border-collapse: collapse` для прибирання подвійних рамок між комірками
- Стилізувати рядок заголовка `<thead>`: задати колір фону та тексту
- Додати чергування кольору рядків за допомогою псевдокласу `:nth-child(even)` або `:nth-child(odd)`
- Задати `padding` для комірок `<td>` та `<th>`
- Додати ефект підсвічування рядка при наведенні за допомогою `tr:hover`

---

#### **Завдання 5: Стилізування структурних блоків**

Оформіть `<header>`, `<section>` та `<footer>` сторінки.

**Вимоги:**

- `<header>` — задати колір фону з CSS-змінної, відступи, вирівняти заголовок по центру, задати колір тексту
- `<section>` — задати відступи між секціями; за бажанням додати роздільну лінію (`border-bottom`)
- `<footer>` — задати темний колір фону, відступи, вирівняти текст по центру, зменшити розмір шрифту відносно базового

---

### **Етап 3: Перевірка**

3.1. Відкрийте `index.html` у браузері та переконайтесь, що стилі застосовуються коректно до всіх елементів.

3.2. Відкрийте DevTools (F12) → вкладка **Elements**. Виберіть будь-який елемент і перегляньте панель **Styles** — ви маєте бачити, які CSS-правила до нього застосовано. Перекриті правила будуть відображені закресленими.

3.3. У вкладці **Styles** перевірте, що CSS-змінні оголошені коректно: виберіть елемент `<html>` або `<body>` і знайдіть у панелі Styles блок `:root { }` — там мають відображатись усі оголошені змінні.

3.4. Перевірте валідність CSS-файлу за допомогою онлайн-валідатора: https://jigsaw.w3.org/css-validator/#validate_by_input (вставте вміст `style.css` і натисніть «Check»).

**Результат:** HTML-сторінка зі стилізуванням через зовнішній CSS-файл, що використовує CSS-змінні, різні типи селекторів та псевдокласи, властивості шрифтів, кольорів і тексту.

## **Типові помилки та їх вирішення**

| Проблема                                 | Рішення                                                                                                                |
| ---------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Стилі не застосовуються до сторінки      | Перевірте підключення `<link>` у `<head>` та правильність шляху до файлу `style.css`                                   |
| CSS-змінна не працює                     | Переконайтесь, що змінна оголошена у `:root {}` і починається з двох дефісів `--`                                      |
| Правило не діє, хоча написане правильно  | Відкрийте DevTools → Styles і знайдіть, яке правило має вищу специфічність і перекриває                                |
| Шрифт не змінюється на всій сторінці     | Задайте `font-family` для `body` — він успадковується всіма вкладеними елементами                                      |
| Таблиця має подвійні рамки між комірками | Додайте `border-collapse: collapse` до правила для `<table>`                                                           |
| Стиль `:hover` не спрацьовує             | Переконайтесь, що елемент має `display` відмінний від `inline` або що псевдоклас застосований до правильного селектора |
