## **Теоретичні відомості**

### **Підключення JavaScript до HTML**

JavaScript підключається до HTML-сторінки за допомогою тегу `<script>`. Рекомендований спосіб — зовнішній файл, розміщений перед закриваючим тегом `</body>`:

```html
<!DOCTYPE html>
<html lang="uk">
  <head>
    <meta charset="UTF-8" />
    <title>Моя сторінка</title>
  </head>
  <body>
    <!-- Контент сторінки -->

    <!-- Підключення JS перед закриттям body -->
    <script src="script.js"></script>
  </body>
</html>
```

### **Змінні: `let` та `const`**

```javascript
// const — значення не можна перевизначити після ініціалізації
const siteName = "Мій сайт";
const MAX_SCORE = 100;

// let — значення можна змінювати
let currentScore = 0;
let isFinished = false;

currentScore = 42; // ✅ Допустимо
// MAX_SCORE = 50;  // ❌ TypeError
```

> **Правило:** надавайте перевагу `const` за замовчуванням. Використовуйте `let` лише тоді, коли значення точно зміниться. `var` не використовується в сучасному JavaScript.

### **Типи даних**

JavaScript має п'ять основних примітивних типів:

| Тип         | Приклади                             | Опис                            |
| ----------- | ------------------------------------ | ------------------------------- |
| `number`    | `42`, `3.14`, `NaN`, `Infinity`      | Цілі та дробові числа           |
| `string`    | `'текст'`, `"рядок"`, `` `шаблон` `` | Послідовність символів          |
| `boolean`   | `true`, `false`                      | Логічне значення                |
| `undefined` | `undefined`                          | Змінна оголошена, але не задана |
| `null`      | `null`                               | Явна відсутність значення       |

```javascript
const age = 20; // number
const name = "Марія"; // string
const isActive = true; // boolean
let result; // undefined

console.log(typeof age); // 'number'
console.log(typeof name); // 'string'
```

### **Шаблонні рядки (Template Literals)**

```javascript
const studentName = "Іван";
const score = 85;

// Конкатенація через + — незручно
console.log("Студент " + studentName + " отримав " + score + " балів.");

// Шаблонний рядок — читабельніше
console.log(`Студент ${studentName} отримав ${score} балів.`);
```

### **Умовні конструкції**

```javascript
const score = 75;

// if / else if / else
if (score >= 90) {
  console.log("Відмінно!");
} else if (score >= 75) {
  console.log("Добре!");
} else if (score >= 60) {
  console.log("Задовільно");
} else {
  console.log("Незадовільно");
}

// switch — для конкретних значень
const day = "понеділок";
switch (day) {
  case "субота":
  case "неділя":
    console.log("Вихідний!");
    break;
  default:
    console.log("Робочий день");
}

// Тернарний оператор
const status = score >= 60 ? "Зараховано" : "Не зараховано";
```

### **Цикли**

```javascript
// for — коли кількість ітерацій відома
for (let i = 1; i <= 5; i++) {
  console.log(`Крок ${i}`);
}

// while — умова перевіряється до виконання тіла
let count = 0;
while (count < 3) {
  console.log(`count = ${count}`);
  count++;
}

// for...of — перебір масиву
const grades = [80, 92, 67, 74, 88];
for (const grade of grades) {
  console.log(grade);
}
```

### **Виведення даних та налагодження**

```javascript
// console.log — основний інструмент розробника
console.log("Звичайне повідомлення");
console.warn("⚠️ Попередження");
console.error("❌ Помилка");

// Діалогові вікна (лише для навчальних цілей)
alert("Повідомлення для користувача");
const answer = confirm("Ви впевнені?"); // true або false
const input = prompt("Введіть ваше ім'я:"); // рядок або null
```

> **Примітка:** `prompt()` завжди повертає **рядок**. Щоб використовувати введене значення як число, перетворіть його явно: `Number(prompt("Введіть число:"))`.

## **Типові помилки та їх вирішення**

| Проблема                                                        | Рішення                                                                                                  |
| --------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Консоль DevTools порожня після відкриття `index.html`           | Переконайтесь, що шляхи до `task1.js`, `task2.js`, `task3.js` вказані правильно у `<script src="...">`   |
| `prompt()` повертає `null`                                      | Так відбувається, якщо користувач натиснув «Скасувати». Додайте перевірку: `if (input !== null) { ... }` |
| Результат арифметики з `prompt` виглядає як «1010» замість «20» | `prompt()` повертає рядок; перетворіть його: `Number(prompt(...))` або `+prompt(...)`                    |
| Цикл `while` виконується нескінченно                            | Перевірте, що змінна-лічильник змінюється всередині тіла циклу (`count++` або `i++`)                     |
| `const` оголошена, але потім перезаписується                    | `const` не допускає повторного присвоєння. Якщо значення зміниться — замініть на `let`                   |
| `switch` не спрацьовує на потрібний `case`                      | Перевірте точний збіг типу і значення — `switch` використовує строге порівняння (`===`)                  |
