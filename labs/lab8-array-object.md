## **Теоретичні відомості**

### **Масиви та основні методи**

Масив — впорядкована колекція значень, індексована з `0`. Для маніпуляції елементами JavaScript надає вбудовані методи:

- `push(item)` / `pop()` — додати або видалити елемент з кінця масиву
- `unshift(item)` / `shift()` — додати або видалити елемент з початку масиву
- `indexOf(value)` / `includes(value)` — пошук елемента

### **Методи вищого порядку**

Ці методи приймають функцію-колбек і є основним інструментом обробки колекцій:

| Метод     | Що повертає                 | Мутує оригінал |
| --------- | --------------------------- | :------------: |
| `forEach` | `undefined`                 |       ❌       |
| `map`     | Новий масив тієї ж довжини  |       ❌       |
| `filter`  | Новий масив (підмножина)    |       ❌       |
| `find`    | Перший збіг або `undefined` |       ❌       |
| `some`    | `boolean`                   |       ❌       |
| `every`   | `boolean`                   |       ❌       |
| `reduce`  | Будь-яке одне значення      |       ❌       |

```javascript
const nums = [3, 7, 2, 9, 4];

const doubled = nums.map((n) => n * 2); // [6, 14, 4, 18, 8]
const big = nums.filter((n) => n > 4); // [7, 9]
const first = nums.find((n) => n > 4); // 7
const sum = nums.reduce((acc, n) => acc + n, 0); // 25
```

### **Об'єкти**

Об'єкт — структура даних «ключ : значення». Ключами є рядки, значеннями — будь-який тип.

```javascript
const sensor = { id: 1, type: "температура", value: 23.5, active: true };

// Доступ до властивостей
console.log(sensor.type); // 'температура'
console.log(sensor["value"]); // 23.5

// Додавання та видалення
sensor.unit = "°C";
delete sensor.active;
```

### **Методи Object**

```javascript
const obj = { a: 1, b: 2, c: 3 };

Object.keys(obj); // ['a', 'b', 'c']
Object.values(obj); // [1, 2, 3]
Object.entries(obj); // [['a', 1], ['b', 2], ['c', 3]]

// Ітерація через властивості
for (const [key, value] of Object.entries(obj)) {
  console.log(`${key} = ${value}`);
}
```

### **Деструктуризація**

```javascript
// Масиву
const [first, second, ...rest] = [10, 20, 30, 40];

// Об'єкта
const { name, age = 18 } = { name: "Олена" };

// У параметрах функції
function greet({ name, city }) {
  return `${name} з ${city}`;
}
```

## **Типові помилки та їх вирішення**

| Проблема                                    | Рішення                                                                                                                     |
| ------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| `map` повертає масив з `undefined`          | Перевірте, що колбек містить явний `return` або використовується скорочений синтаксис стрілкової функції без фігурних дужок |
| `reduce` кидає помилку на порожньому масиві | Завжди передавайте `initialValue` другим аргументом: `reduce(cb, 0)`                                                        |
| `find` повертає `undefined`                 | Перевірте умову в колбеку — можливо, жоден елемент не відповідає критерію                                                   |
| Оригінальний масив змінюється після `sort`  | `sort` мутує масив; створюйте копію через `[...arr].sort(...)`                                                              |
| Spread не виконує глибоку копію             | `{...obj}` копіює лише перший рівень; вкладені об'єкти залишаються спільними посиланнями                                    |
| `toFixed` повертає рядок, а не число        | Це очікувана поведінка: `(3.567).toFixed(1)` → `'3.6'` (тип `string`)                                                       |
